Notes on False Consciousness

Suppose a person “behaves as he does because of a system or socially institutionalized set of behaviors that are entrenched in the society, and that the function of the system” is sometimes opaque/not completely visible to the actor.

The actor in this situation exhibits false consciousness, because of the gap between what he thinks he’s doing (and why) and what he’s actually doing (and why).

  • He thinks: he’s being kind / chivalrous.
  • Actually: he’s perpetuating a system with the function to demean women.
